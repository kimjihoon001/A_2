------------------------------------------------------------
[1] 우리 노드 (robot_art_node) — 직접 정의한 통신
------------------------------------------------------------

■ Publisher (발행)
  토픽명                  타입              설명
  /robot_art/status       std_msgs/String   로봇 상태 JSON 주기 발행 (0.5초마다)
  /robot_art/log          std_msgs/String   로그 메시지 발행

■ Subscription (구독)
  토픽명                  타입              설명
  /robot_art/pixels       std_msgs/String   픽셀 데이터 수신 (JSON, 그리기 시작 전 전송)

■ Service Server (제공)
  서비스명                          타입                    설명
  /robot_art/start                  std_srvs/Trigger        그리기 시작
  /robot_art/stop                   std_srvs/Trigger        강제 정지
  /robot_art/pause                  std_srvs/Trigger        일시정지
  /robot_art/resume                 std_srvs/Trigger        재개
  /robot_art/estop                  std_srvs/Trigger        E-STOP 활성화
  /robot_art/release_estop          std_srvs/Trigger        E-STOP 해제
  /robot_art/home                   std_srvs/Trigger        홈 복귀
  /robot_art/gripper_open           std_srvs/Trigger        그리퍼 열기
  /robot_art/gripper_close          std_srvs/Trigger        그리퍼 닫기
  /robot_art/pencil_grip            std_srvs/Trigger        연필 파지
  /robot_art/pencil_release         std_srvs/Trigger        연필 반납
  /robot_art/calibrate_z            std_srvs/Trigger        Z축 자동 캘리브레이션
  /robot_art/paper_check            std_srvs/Trigger        종이 감지 확인
  /robot_art/confirm_retry          std_srvs/Trigger        오류 발생 시 재시도 확인
  /robot_art/frame_task             std_srvs/Trigger        액자 작업 전체 실행
  /robot_art/frame_lower            std_srvs/Trigger        액자 하판 배치
  /robot_art/frame_paper_pickup     std_srvs/Trigger        종이 픽업
  /robot_art/frame_align            std_srvs/Trigger        종이 정렬
  /robot_art/frame_upper            std_srvs/Trigger        액자 상판 배치
  /robot_art/frame_eject            std_srvs/Trigger        액자 배출


------------------------------------------------------------
[2] 백엔드 (main.py / robot_controller.py) — DSR 직접 통신
------------------------------------------------------------

■ Publisher (발행)
  토픽명                  타입              설명
  /robot_art/pixels       std_msgs/String   픽셀 데이터 전송 (→ robot_art_node 구독)

■ Subscription (구독)
  토픽명                        타입                          설명
  /dsr01/msg/joint_state        std_msgs/Float64MultiArray    관절 각도 실시간 수신
  /dsr01/msg/current_posx       std_msgs/Float64MultiArray    TCP 위치 실시간 수신
  /robot_art/status             std_msgs/String               robot_art_node 상태 수신
  /robot_art/log                std_msgs/String               robot_art_node 로그 수신

■ Service Client (호출)
  서비스명                                타입                          설명
  /robot_art/{name}                       std_srvs/Trigger              robot_art_node 서비스 호출 (위 목록 전체)
  /dsr01/system/servo_off                 dsr_msgs2/ServoOff            서보 OFF (E-STOP 시 토크 차단)
  /dsr01/system/set_robot_control         dsr_msgs2/SetRobotControl     서보 ON / Safe Stop 리셋
  /dsr01/system/get_robot_state           dsr_msgs2/GetRobotState       로봇 상태 폴링 (0.5초마다)
  /dsr01/system/set_safe_stop_reset_type  dsr_msgs2/SetSafeStopResetType Safe Stop 해제 방식 설정 (초기화 시 1회)
  /dsr01/motion/move_stop                 dsr_msgs2/MoveStop            모션 즉시 정지
  /dsr01/motion/jog                       dsr_msgs2/Jog                 단축 조그 이동
  /dsr01/motion/jog_multi                 dsr_msgs2/JogMulti            다축 조그 이동
  /dsr01/system/set_robot_mode            dsr_msgs2/SetRobotMode        로봇 모드 설정
